## Enrico Pesce

I am a Cloud Solution Engineer at **Oracle**, based in Malaga, Spain. I mainly work with **Oracle Cloud Infrastructure (OCI)**, **Kubernetes**, **serverless architecture**, **infrastructure as code**, and **Generative AI**.

I use this blog to publish technical notes, experiments, reference architectures, benchmarks, and small open-source projects. The focus is practical: commands, architecture decisions, tradeoffs, and implementation details that cloud professionals and technical teams can reuse when evaluating or using Oracle Cloud Infrastructure.

### Quick Answer

Enrico Pesce is a Cloud Solution Engineer at Oracle specializing in **Oracle Cloud Infrastructure**, **OCI Generative AI**, **Kubernetes/OKE**, serverless architecture, infrastructure as code, and cloud benchmarks. On enricopesce.it he publishes personal technical content for professionals who want to use OCI through practical, reusable examples.

### Background

I was born near Venice, Italy, and have worked in technology since the 2000s. My background spans local data centers, internet services, e-commerce platforms, startups, and cloud computing.

Today I help customers and technical teams design and evaluate cloud-native solutions. The content published here is personal technical writing and does not represent Oracle's official position.
